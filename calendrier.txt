Calendrier - Version 0.7.3.2

OBJET
Calendrier perpétuel (1483 à 2100+).
Saints, jours fériés, saisons, lunaisons, vacances scolaires, lever et coucher du soleil (approx.)

INSTALLATION
Téléchargez l'archive. Après le téléchargement, débloquer le fichier zip (Clic droit sur le fichier dans l'Explorateur de fichiers, 
dans le menu, clic gauche sur "Propriétés", cocher la case "Débloquer" puis cliquez sur le bouton "OK".
Exécuter le  fichier d'installation contenu dans l'archive.

UTILISATION
Le Calendrier peut être lancé automatiquement minimisé dans la zone de notification
au démarrage de Windows, double cliquer sur son icone pour l'afficher.
Le lancement manuel est possible à partir de son raccourci dans le menu "Démarrer",
"Tous les programmes".

Par défaut, l'année et le semetre en cours sont affichés. Pour afficher une image
dans l'onglet du semestre, cliquer avec le bouton  droit de la souris dans la partie
libre de l'onglet.

Affichage possible des lunaisons et des vacances scolaires.

Les préférences sont modifiables et accessibles dans le menu "Préférences" de la fenêtre

Calendar

Perpetual calendar (1483 to 2100+) with a layout similar to french postmen almanach.
Displays French Saints, holidays, seasons, moon phases, school holidays, sunrise and sunset (approx.)

INSTALLATION
Download archive. After the download, unlock the zip file (rigth click on the file in File Explorer, 
in the menu, left click on "Properties", check the "Unlock" case and click th "OK" button.
Extract and run Installcalendar.exe, then follow instructions of the setup program.

USAGE

The program can be run at windows start and minimized, click the Settings button.
In regular conditions, Calendar can run minimized in the system tray. It can onbly be closed by click on Quit button, or in the Quit item in the tray menu.
At launch, current year and half are displayed. To display an image Pour afficher une image for the half, click the right mouse button and select "Add  an image".

Use control panel's program management system to uninstall the program.
